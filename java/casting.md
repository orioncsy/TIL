# Casting

## 개념

### casting

- 형변환이라고도 하며 다른 타입의 정보도 가지고 있어 변환 가능할 경우 사용

### 목적

- 다형성을 통해 오버라이딩 된 함수를 분리해서 활용 가능
- 상속을 통해 프로그래밍 가능

## 종류

### 묵시적 형변환

- 캐스팅이 자동으로 발생
- 업캐스팅에서 사용
    - 상위 클래스에게 하위 클래스 객체를 할당하는 경우
    - 형변환 기호를 붙이지 않아도 된다.

### 명시적 형변환

- 캐스팅 내용을 직접 작성
- 다운 캐스팅에서 사용
    - 상위 클래스의 객체를 하위 클래스 형태로 변환
    - 소괄호를 통해 구현
